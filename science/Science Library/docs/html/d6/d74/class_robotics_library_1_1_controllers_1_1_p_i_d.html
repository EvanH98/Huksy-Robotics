Detailed Description

Class built to create a PID control loop in python Based off of the PID control algorithm: https://en.wikipedia.org/wiki/PID_controller

  • * Built for Real-Time Operation * * Basic Implementation as follows: 1) Choose P, I, and D constants in constructor 2) Set a target value using SetTarget() (Units do not matter as long as they are used uniformly in the loop) 3) Use Feed(...) to run the PID loop given a sensor reading (Should be called iteratively, since it depends on system time) 4) Use YourPIDInstance.Output to return the desired output (Output restrained to +/-1, tune PID coefficients accordingly)
  • NOTE: This output will be relative to the units given in the Feed() method. Map this output to a given range if necessary, for example, in using motors, consider a map to -1 to 1
  • NOTE: Nothing has to be done to change the target value during operation, it is all handled in this class.
  • * IMPORTANT * * A PID Control loop needs to be tuned. That is, to be used correctly, the P, I, and D coefficients need to be precisely chosen. An algorithm for determining coefficients: https://en.wikipedia.org/wiki/PID_controller#Ziegler.E2.80.93Nichols_method
