Detailed Description

The Average filter is intended for use as an average-gathering system, using a rolling average with "roll-length" FilterCount.

Implementation Details:

*Construct Average filter given a rolling filter length, FilterCount

*Iteratively add values into the filter using Feed(T Input)

*Get the filter output by calling YourFilterInstance.Output
